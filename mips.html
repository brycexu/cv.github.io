MIPS Microprocessor

Description

  • Xianda Xu

  • Microprocessor Design (UESTC)

It is a MIPS Single Pipeline Microprocessor that runs on FPGA. It has a fast and accurate Arithmetic Logical Unit and Hazard Control scheme. (50M for FFT-8 Operations)
